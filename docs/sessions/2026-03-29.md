## Technical Notes

- Calendar parses events from the hidden list view DOM (no duplicate data source)
- Ticker tape uses `width: max-content` + flex + `cloneNode` for seamless loop
- Content repeated dynamically until wider than 2x viewport to prevent gaps
- Animation speed calculated as px/sec for consistent speed across viewports
- Cost counter uses `requestAnimationFrame` for smooth updates
- Accordion uses native `<details>` elements for accessibility and zero JS
